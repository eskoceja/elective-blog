NETWORK INFRASTRUCTURE BASICS

"Network infrastructure refers to the hardware and software that enable network connectivity and communication between users, devices, apps, the internet, and more. "
- CISCO
  • IP Addresses (Internet Protocol Address): A unique string of characters that identifies each device and can be used for devices to communicate with each other.
  • Subnet mask: A 32-bit value used with IP addresses to divide a network into subnetworks(subnets). It helps identify which part of the IP address represents the network and which part represents the host.
  • Default gateway: The IP address of the router or gateway device that connects a LAN to external networks.
  • DNS (Domain Name System): A system that translates domain names into IP addresses. It allows users to visit websites and other netowrk resources using human-readable domain names.
  • LAN (Local Area Network): A network that connects devices within a limited area, like a home, office, or school. It allows the devices to share resources, like printers and internet access.
  • WAN (Wide Area Network): A network that covers a large geographical area, usually connecting multiple LANs together. It allows for long-distance communication and connection.
  • Switch: A networking device that connects multiple devices within a network and eases communication between them. It operates in the data link layer of the network protocol and forwards data packets to their intended destination based on MAC addresses.
  • Router: A networking device that connects multiple networks together and routes data packets between them. It operates in the network layer of the network protocol and uses IP addresses to determine the most efficient path to transmit data.
  • Bandwidth: Max amount of data that can be transmitted over a network connectiong withing a given period. It is usually measured in bits per second (bps) and represents the capacity of the network connection.
  • Latency: Amount of time it takes for the data packets to travel from the source to the destination across a netowrk. Often measured in milliseconds (ms) and represents the delay or responsiveness of the network connection.
