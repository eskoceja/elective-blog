What is R?

The programming language R is used for statistical computing and graphics. R provides a variety of statistical and graphical techniques such as: linear and nonlinear modeling, classical statistical tests, time series analysis, classification, etc. One of the strengths of this language and environment is how easy a well-designed publication quality plot can be produced. It can even include mathematical symbols and formulas as needed.

R is an free open-source code that runs on a wide variety of UNIX platforms and systems similar to it. Because of this, the community around R have kept it up-to-date and the community continues to grow as people want to contribute to its progression.

Some of the features include:

  • Effective data handling and ease of storing it
  • Suite of operators for calculations on data collections
  • Variety of tools for data analysis
  • Ease to graphically display data analysis on screen and hard copy
  • Includes conditionals, loops, user defined recursive functions, and input/output facilities

In the Udemy course I learned a great deal of things on R more closely related to data analytics. To create a new script, it has to have the ".R" extension that specifies we are working with that programming language. There are also like four different ways to create an object (myobject = 5:10, myobject <- 5:10, 5:10 -> myobject, and assign("myobject", 5:10)). In that particular example, the 5:10 indicates that the object consists of integers 5-10. To plot that data, you can use a built in function called plot(myobject). These are very basic features. I also learned that R can preform data entries, data pre-processing, statistical analysis, data simulations, data visualizations, web scraping, and data visualization with web integration. What is PowerShell?
"PowerShell is a cross-platform task automation solution made up of a command-line shell, a scripting language, and a configuration management framework. PowerShell runs on Windows, Linux, and macOS." - Microsoft

PowerShell is a modern command-line shell that accepts and returns .NET objects. Some of it's features consist of a robust command-line history, tab completion and command prediction, supports parameter aliases, pipelines for chaining commands, and in-console help system.

PowerShell is most commonly used for automation and management of systems as it is a scripting language. It can also be used to build, tests, and the deployment of solutions, often in CI/CD environments.
